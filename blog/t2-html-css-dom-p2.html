Q: What is grid based design?
A: A grid based design divides the whole webpage into rows containing columns. Alot of web developers use 12 columns

Q: Whats all the hype about responsive webpages?
A: Responsive webpage means that whatever device your on it'll adapt to your screen. It also allows you to use less code instead of typing alot of different files for different size devices you only need to write one set of code.

Q: What is semantic structure?
A: A semantic structure is the way the information is laid out in the semantic tags which are "< >" inbetween the semantic tags is where you put you information or how you lay the webpage out for css.
eg. < h1 > < title > < div > < head > < body >
